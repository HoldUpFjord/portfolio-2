## Overview

Gridiron Survivor is a NFL Survivor Pool product. Built on NextJS, Appwrite, and ShadCN, GridIron Survivor delivers a modern NFL Survivor Pool experience.

## Key Features

- **Team based process **: Collaborated with a cross-functional team of 10, including developers and designers, to improve UX and test coverage.

- **Expanded Test Coverage**: Expanded Unit Test coverage by 15% in Jest.

- **Developed a reusable Sidebar Component**: Refactored sidebar list items into reusable components.

## Technologies Used

- **NextJS**: For dynamically rendering content on the client side and statically rendering on the server.

- **Appwrite**: A backend as a service platform that runs on PostgreSQL.

- **ShadCN**: Styling library that is built on Radix Primitives with TailwindCSS.

- **Jest and Playwright**: For unit and end-to-end testing respectively.

- **Typescript**: For type safety and better code quality.

## Challenges and Learnings

Gridiron Survivor is a distributed team running on AGILE and Scrum principles. Balancing development velocity with rigorous code reviews presented the primary challenge.

## Outcome

Gridiron survivor has onboarded more than 100 users from traditional tools, and it's still in Alpha. Looking forward to expanding the userbase and iterating on the feature set.
